# Multi-Strategy Heart Disease Risk Analysis 🩺📊

An advanced data science project utilizing the Heart Disease Dataset (50,000+ entries) to identify critical risk factors through 60 unique analytical strategies.

## 🚀 Key Features
- **Feature Importance Ranking:** Using Random Forest to identify primary drivers of cardiac arrest.
- **Hazard Window Mapping:** Identifying the age-habit intersection where risk spikes.
- **Biological Signatures:** Radar plots comparing Healthy vs. Diseased profiles.
- **Data Manifold:** PCA-based visualization of the "Cardiac Galaxy."

## 🛠️ Tech Stack
- **Environment:** Google Colab
- **Languages:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn, Seaborn, Matplotlib

## 📈 Top Insight
Discovered that the combination of High Stress and Sedentary Lifestyle creates a "Double Risk" zone that increases disease probability by over 40% in patients aged 50-60.
